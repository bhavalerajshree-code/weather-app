# Weather App

A simple and interactive weather application that provides real-time weather updates based on user location or city search.

## 🚀 Features
- Get current weather details (temperature, humidity, etc.)
- Search weather by city name
- Real-time API data fetching
- Responsive and user-friendly interface

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- Weather API

## 💡 Highlights
- Integrated API for live weather updates
- Dynamic UI updates using JavaScript
- Clean and responsive design
